Um site simples que utiliza uma API de busca de pokemons, a busca é feita utilizando o nome ou número da pokedex que o pokemon ocupa, ela retorna um card contendo nome, imagem do pokemon (tenta utilizar uma imagem do anime, se não existir pega o sprite padrão), id/número da pokedex, tipagem do pokemon (elementos que ele possui), altura e peso médio da espécie, habilidades especias que o pokemon pode ter no principal jogo que ele aparece. Caso o nome digitado esteja errado ele exibe uma mensagem de erro, caso o input esteja vazio ele irá disparar um alert que o campo está vazio e não fará a pesquisa.
